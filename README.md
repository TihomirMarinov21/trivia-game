# trivia-game
Trivia Game (Python)

A simple console-based trivia game written in Python, featuring two categories: General Knowledge and Computer Science. The game is designed to be played by multiple players, where each player answers a series of questions and earns points. The player with the highest score wins the game.

Features

Two Categories:

General Knowledge

Computer Science

Multiple Players:

Allows 2 or more players to play the game.

Question Rotation:

The game alternates between categories every 7 questions.

Input Validation:

Ensures that only valid answers (A, B, C, or D) are accepted.

Scoreboard:

Displays the score after each round, and declares a winner at the end of the game.
